Scope

Scope in Javascript is very important, and refers to the ability to access functions and variables depending on their position in the code relative to each other. A variable or function can either have local or global scope. If it is global it exists outside of a function, and is data that can be accessed by any other function or variable. Whereas if it is within a function it is local and can only be accessed specifically within that function.
